# 💬 CORECHAT

Chat en tiempo real completo y profesional construido con Flask y Socket.IO, optimizado para PC y móvil.

## ✨ Características
- 🔐 Autenticación segura con contraseñas hasheadas
- 💬 Múltiples salas de chat
- 💌 Mensajes privados
- 👑 Panel de administración completo
- 🖼️ GIFs y stickers integrados
- 📱 Optimizado para móvil
- 💾 SQLite/PostgreSQL (soporte dual)

## 🔑 Credenciales
- **Admin:** `Administrador` / `1234`

## 🛠️ Tecnologías
- Flask 3.1 + Socket.IO
- Python 3.14
- SQLite / PostgreSQL
- HTML5 + CSS3 + JavaScript
